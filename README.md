# 🏟️ StadiumFanFlow

> **AI-Powered Smart Stadium Platform for FIFA World Cup 2026**

StadiumFanFlow is a Generative AI-powered smart stadium platform built to improve the experience of fans, organizers, volunteers, and stadium staff during large-scale sporting events like the FIFA World Cup 2026.

The platform combines AI-driven crowd intelligence, smart navigation, multilingual assistance, operational dashboards, and real-time decision support to make stadium operations safer, smarter, and more efficient.

---

## 🚀 Problem Statement

Managing thousands of spectators during international sporting events presents several challenges:

- Crowd congestion
- Navigation inside large stadiums
- Long queues
- Language barriers
- Accessibility for differently-abled fans
- Emergency response delays
- Operational coordination
- Transportation & parking management

StadiumFanFlow leverages Generative AI to solve these problems in real time.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/StadiumFanFlow.git
```

Move into the project

```bash
cd StadiumFanFlow
```

Install dependencies

```bash
npm install
```

Create a `.env.local`

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
npm run dev
```

Open

```
http://localhost:3000
```
